# ZOJ-a002 - 簡易加法

請寫一個程式，讀入兩個數字，並求出它們的和。

> * 記憶體限制：512 MB
> * 公開 測資點#0 (20%): 1.0s , < 1K
> * 公開 測資點#1 (20%): 1.0s , < 1K
> * 公開 測資點#2 (20%): 1.0s , < 1K
> * 公開 測資點#3 (20%): 1.0s , < 1K
> * 公開 測資點#4 (20%): 1.0s , < 1K

---
## Input

每一組輸入有兩個整數，絕對值皆小於 $10^6$。

---
## Output

對於每組輸入，輸出該兩整數的和。

---
## Sample Input

```
5 10
1 2
```

---
## Sample Output

```
15
3
```
