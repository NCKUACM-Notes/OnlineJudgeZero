# ZOJ-a121 - 質數又來囉

你的好朋友質數先生又來找你囉，給你兩個數字，請算出這兩個數字包含的範圍內有幾個質數。

> * 記憶體限制： 512 MB
> * 公開 測資點#0 (100%): 3.0s , < 1M

---
## Input

輸入兩個正整數 $a$ 和 $b$ ($1 \le a \le b \le 100000000$)。保證 $b-a \le 1000$。

---
## Output

輸出一個非負整數，代表 $a$ 到 $b$ 之間 (包含 $a$，$b$) 總共有幾個質數。

---
## Sample Input

```
3 7
6 6
30 50
```

---
## Sample Output

```
3
0
5
```
