# ZOJ-a001 - 哈囉

學習所有程式語言的第一個練習題。請寫一個程式，可以讀入指定的字串，並且輸出指定的字串。

> * 記憶體限制：512 MB
> * 公開 測資點#0 (20%): 1.0s , < 1K
> * 公開 測資點#1 (80%): 1.0s , < 1M

---
## Input

輸入指定的文字。

---
## Output

輸出指定的文字。

---
## Sample Input

```
world
C++
mary
```

---
## Sample Output

```
hello, world
hello, C++
hello, mary
```
