# ZOJ-a011 - Kindergarten Counting Game

算一算每行有幾個字 (word)。「字（word）」的定義是連續的字元 (字母：A - Z，a - z) 所組成的字。

> * 題目來源：**UVA-494**
> * 記憶體限制： 512 MB
> * 公開 測資點#0 (100%): 1.0s , < 1K

---
## Input 

一段文字 (string)。

---
## Output

字數 (int)。

---
## Sample Input

```
Hello everybody!!
This is school principal speeking.
```

---
## Sample Output

```
2
5
```
