# ZOJ-a104 - 排序

請將數字排序。

> * 記憶體限制： 512 MB
> * 公開 測資點#0 (100%): 1.0s , < 1M

---
## Input

有多筆測資以 `EOF` 為結束。第一行有一個正整數 $n$ ($1 \le n \le 1000$)，代表有幾個數字要請你排序，第二行有 $n$ 個可以用 `int` 儲存的正整數。

---
## Output

輸出 $n$ 個由小到大排序好的正整數。

---
## Sample Input

```
6
7 9 0 4 1 8
8
1 9 9 0 0 9 2 8
```

---
## Sample Output

```
0 1 4 7 8 9
0 0 1 2 8 9 9 9
```
